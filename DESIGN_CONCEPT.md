## ‚ö° ANIMATION GUIDELINES

### Principles
1. **Purposeful**: Setiap animasi harus ada tujuan
2. **Subtle**: Jangan berlebihan, distract user
3. **Fast**: Duration 200-500ms
4. **Smooth**: Gunakan ease functions

### Timing Functions
```css
--transition-fast: 0.2s ease;    /* Hover effects */
--transition-base: 0.3s ease;    /* Standard transitions */
--transition-slow: 0.5s ease;    /* Fade in/out */
```

### Animation Types

#### 1. Fade In Up
**Usage**: Hero content, sections on scroll
```css
@keyframes fadeInUp {
    from {
        opacity: 0;
        transform: translateY(30px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}
```

#### 2. Hover Lift
**Usage**: Cards, buttons
```css
.card:hover {
    transform: translateY(-10px);
    box-shadow: var(--shadow-xl);
}
```

#### 3. Scale on Hover
**Usage**: Icons, images
```css
.icon:hover {
    transform: scale(1.2);
}
```

#### 4. Slide In
**Usage**: Mobile menu, modals
```css
.menu {
    transform: translateX(-100%);
    transition: transform 0.3s ease;
}
.menu.active {
    transform: translateX(0);
}
```

#### 5. Counter Animation
**Usage**: Statistics, numbers
```javascript
// Animate from 0 to target number
// Duration: 2 seconds
// Easing: ease-out
```

#### 6. Parallax Scroll
**Usage**: Hero background
```javascript
// Move slower than scroll speed
// Multiplier: 0.3
```

### Performance Tips
- Use `transform` and `opacity` (GPU-accelerated)
- Avoid animating `width`, `height`, `top`, `left`
- Use `will-change` sparingly
- Debounce scroll events
